## 🚀 Live Demo

[Live Demo](https://awsys.bohdan.lol)

## 📦 Installasjon og oppstart

1. Sørg for at du har Node.js og npm installert.

2. Klon repoet:

```bash
git clone https://github.com/Bogdan3000/AtWorkSystems
cd AtWorkSystems
```

3. Installer avhengigheter:

```bash
npm install
```

4. Start prosjektet i produksjonsmodus:

```bash
npm run start:prod
```

## 📌 Om prosjektet

Dette prosjektet er en moderne nettside for At Work Systems, bygget med React (Frontend) og Node.js (Backend).

✨ Designet har et mørkt, minimalistisk tema som fremhever produkter og inkluderer interaktive spørreundersøkelser og flytskjemaer.
